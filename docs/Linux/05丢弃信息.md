system执行$c的终端命令 **">/dev/null 2>&1"** 加上这句后，执行命令但无回显

> 在Linux命令行中，分号（;）、竖线（|）、双竖线（||）、和符号（&）以及双与符号（&&）有不同的作用：

```bash
分号（;）：用于分隔多个命令，依次执行每个命令，不管前面的命令是否成功。
竖线（|）：管道符号，用于将一个命令的输出传递给另一个命令作为输入。
双竖线（||）：逻辑或运算符，用于在第一个命令失败时执行第二个命令。
和符号（&）：在命令末尾使用表示在后台执行该命令。
双与符号（&&）：逻辑与运算符，用于只有在第一个命令成功执行后才执行第二个命令。
```

**其中在使用GET传值时，&和&&都需要进行url编码**
