# SSRF

> SSRF(Server-Side Request Forgery,服务器请求伪造)是一种由攻击者构造请求,由服务端发起请求的安全漏洞,一般情况下,SSRF攻击的目标是外网无法访问的内网系统

## 本地IP地址的各种进制

#默认
http://127.0.0.1

#16进制
http://0x7F000001

#10进制
((127*256+0)*256+0)*256+1//计算过程
http://2130706433

#8进制
http://0177.0000.0000.0001


## 0

0.0.0.0表示任何IP地址，也可以表示本地

http://0.0.0.0

http://0.0.0

http://0.0

http://0
