# [极客大挑战 2019]Secret File

在Archive_room.php里的按钮点击后，跳转到了end.php。在调试工具网络里可以看到，这个过程请求了action.php，但是又跳到end.php了

一时手足无措了。看了大佬思路，发现可以bp抓包看这个过程的action.php，又学到了

抓包可以看到action.php页面有提示文件secr3t.php文件
